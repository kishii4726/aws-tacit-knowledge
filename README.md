# awsselfrev

## Usage
```
$ awsselfrev ecr
+---------+---------+------------------------------------------------------+
| SERVICE |  LEVEL  |                       MESSAGE                        |
+---------+---------+------------------------------------------------------+
| ECR     | Warning | repository1のイメージのスキャンが有効になっていません    |
| ECR     | INFO    | repository1にライフサイクルポリシーが設定されていません  |
| ECR     | Warning | repository2のイメージのスキャンが有効になっていません    |
| ECR     | INFO    | repository2にライフサイクルポリシーが設定されていません  |
+---------+---------+------------------------------------------------------+

$ awsselfrev s3
+---------+---------+----------------------------------------------------------------------------------------------------------------------+
| SERVICE |  LEVEL  |                                                       MESSAGE                                                        |
+---------+---------+----------------------------------------------------------------------------------------------------------------------+
| S3      | Warning | bucket1のパブリックブロックアクセスがすべてオフになっています                                                             |
| S3      | Alert   | bucket2のが暗号化されていません                                                                                        |
| S3      | Warning | bucket3-logにライフサイクルルールが設定されていません                                                                    |
+---------+---------+----------------------------------------------------------------------------------------------------------------------+
```
